welcome to the campaign party!

here I'm keeping some resources that I like to use with my games.

maps

  • dave's mapper: a fun tilebased map-maker
  • wonderdraft: an excellent world map-maker
  • donjon: particularly a fractal world generator and a fantasy dungeon generator

organization

  • airtable: spreadsheets (5E character creation options, spell compendium, campaign tracker template, etc)
  • homebrewery: write your own documents that look like a source-book!

command-line

if regular tabletop roleplaying isn't nerdy enough, have some half-baked code projects!
beware: these may be unsupported, unfinished, or ungood
  • dice: a CLI tool in Go that parses die strings like "1d20" or "4d6 drop lowest"
  • randomly-generated magic items: a CLI tool in Python that helps you come up with random magic items
  • randum dungeon generator: a macOS application that helps you generate dungeons
